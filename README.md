# Plot SPECT image

```
Usage: plot_spect_image.py [-h] [--outputfile OUTPUTFILE] inputfile

positional arguments:
  inputfile             A datafile created by the old SPECT camera

optional arguments:
  -h, --help            show this help message and exit
  --outputfile OUTPUTFILE, -o OUTPUTFILE
                        If given, save the image to outputfile
```
